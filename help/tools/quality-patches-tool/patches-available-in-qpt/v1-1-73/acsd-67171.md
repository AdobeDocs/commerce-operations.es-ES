# ACSD-67171: la sesión [!DNL B2B] caducada durante la desprotección redirige a la página Acceso denegado en lugar de al inicio de sesión

La revisión ACSD-67171 corrige un problema en el cual se redirige a los clientes de [!DNL B2B] a una página de *Acceso denegado* cuando la sesión caduca o se elimina durante el cierre de compra. Esta revisión está disponible cuando está instalado [[!DNL Quality Patches Tool (QPT)]](/help/tools/quality-patches-tool/quality-patches-tool-to-self-serve-quality-patches.md) 1.1.73. El ID del parche es ACSD-67171. Este problema está programado para solucionarse en Adobe Commerce 2.4.9.

## Productos y versiones afectados

**El parche se ha creado para la versión de Adobe Commerce:**

* Adobe Commerce (todos los métodos de implementación) 2.4.8-p1

**Compatible con versiones de Adobe Commerce:**

* Adobe Commerce (todos los métodos de implementación) 2.4.4 - 2.4.8-p3

>[!NOTE]
>
>El parche podría ser aplicable a otras versiones con las nuevas versiones de [!DNL Quality Patches Tool]. Para comprobar si el parche es compatible con su versión de Adobe Commerce, actualice el paquete `magento/quality-patches` a la última versión y compruebe la compatibilidad en la página [[!DNL Quality Patches Tool]: buscar parches ](https://experienceleague.adobe.com/tools/commerce-quality-patches/index.html). Utilice el ID de parche como palabra clave de búsqueda para localizar el parche.

## Problema

El sistema muestra una página Acceso denegado durante el cierre de compra cuando la sesión de un usuario de [!DNL B2B] caduca o se elimina, en lugar de redirigirlo a la página de inicio de sesión del cliente.

<u>Pasos a seguir</u>:

1. En el panel de administración, vaya a [!UICONTROL Stores] > [!UICONTROL Configuration] > [!UICONTROL General] > [!UICONTROL [!DNL B2B] Features], establezca [!UICONTROL [!DNL B2B] Features] en *Sí*, después vaya a [!UICONTROL Customers] > [!UICONTROL Persistent Shopping Cart] y establezca [!UICONTROL Enable Persistence] en *Sí*.
1. Vaya a [!UICONTROL Customers] > [!UICONTROL Companies] y seleccione [!UICONTROL Add New Company] para crear una nueva compañía de [!DNL B2B].
1. Inicie sesión en la tienda como el usuario de la compañía [!DNL B2B].
1. Añada cualquier producto al carro de compras.
1. Continúe con la página Cierre de compra.
1. Permita que la sesión caduque.
1. Vuelva a cargar la página Cierre de compra.

<u>Resultados esperados</u>:

Se le redirigirá a la página de inicio de sesión del cliente.

<u>Resultados reales</u>:

Se le redirigirá a `/company/accessdenied` y verá *No tiene permisos para ver esta página. Si cree que se trata de un error, póngase en contacto con el administrador de la empresa.*
