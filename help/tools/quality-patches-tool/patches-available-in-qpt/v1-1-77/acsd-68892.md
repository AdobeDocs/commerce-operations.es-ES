# ACSD-68892: comportamiento de almacenamiento en caché [!DNL Fastly] incoherente para páginas almacenables en caché

La revisión ACSD-68892 corrige el problema en el que las páginas almacenables en caché no se almacenan o sirven correctamente desde la caché [!DNL Fastly], lo que da como resultado un comportamiento de almacenamiento en caché incoherente y un rendimiento reducido. Esta revisión está disponible cuando está instalado [[!DNL Quality Patches Tool (QPT)]](/help/tools/quality-patches-tool/quality-patches-tool-to-self-serve-quality-patches.md) 1.1.77. El ID del parche es ACSD-68892. Este problema está programado para solucionarse en Adobe Commerce 2.4.9.

## Productos y versiones afectados

**El parche se ha creado para la versión de Adobe Commerce:**

* Adobe Commerce (todos los métodos de implementación) 2.4.8-p2

**Compatible con versiones de Adobe Commerce:**

* Adobe Commerce (todos los métodos de implementación) 2.4.8 - 2.4.8-p3

>[!NOTE]
>
>El parche podría ser aplicable a otras versiones con las nuevas versiones de [!DNL Quality Patches Tool]. Para comprobar si el parche es compatible con su versión de Adobe Commerce, actualice el paquete `magento/quality-patches` a la última versión y compruebe la compatibilidad en la página [[!DNL Quality Patches Tool]: buscar parches &#x200B;](https://experienceleague.adobe.com/tools/commerce-quality-patches/index.html). Utilice el ID de parche como palabra clave de búsqueda para localizar el parche.

## Problema

Las páginas almacenables en caché no se almacenan ni sirven correctamente desde la caché de [!DNL Fastly], lo que da como resultado un comportamiento de almacenamiento en caché incoherente y un rendimiento reducido.

<u>Pasos a seguir</u>:

1. Habilitar la caché [!DNL Fastly].
1. Crear una nueva vista de sitio web, tienda o tienda.
1. Agregar el código del sitio web secundario a `index.php`:

```
$_SERVER['MAGE_RUN_TYPE'] = 'website';
$_SERVER['MAGE_RUN_CODE'] = 'web2';
```

1. Compruebe el encabezado de control de caché con curl:

```
curl -Is https://localhost/m247/pub/ | grep cache-control
```

<u>Resultados esperados</u>:

Las páginas almacenables en caché se almacenan en caché en [!DNL Fastly].

<u>Resultados reales</u>:

Las páginas almacenables en caché no se almacenan en caché en [!DNL Fastly], `max-age=0` devolvió en el encabezado.

```
curl -Is https://localhost/m24develop/pub/ |grep cache-control
cache-control: max-age=0, must-revalidate, no-cache, no-store
```
